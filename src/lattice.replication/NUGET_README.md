# Orleans.Lattice.Replication

Cross-cluster replication for [Orleans.Lattice](https://www.nuget.org/packages/Orleans.Lattice).

This package layers on top of `Orleans.Lattice` and ships the on-the-wire replication engine for moving mutations between Orleans clusters. See the [project README](https://github.com/NSTA1/Orleans.Lattice) for full documentation.
